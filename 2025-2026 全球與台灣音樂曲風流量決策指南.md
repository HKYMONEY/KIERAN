# 2025-2026 全球與台灣音樂曲風流量決策指南

> **核心模型**：多維度綜合加權降序模型（綜合原始播放量、聽眾主動收藏率、市場增長動能、轉媒體影響力）
> **資料來源**：Spotify / Apple Music / RouteNote 公開資料 + Suno v5.5 內部測試
> **覆蓋範圍**：全球 + 台灣市場流量權重 Top 50 + RouteNote 39 個合法曲風映射

---

## 📊 一、權重模型核心邏輯

| 權重因素 | 說明 | 高分代表 |
|----------|------|----------|
| **Save Rate（收藏率）** | 聽眾主動收藏行為 | Country / Rock / K-Pop / Christian |
| **Super Fan 黏著度** | 重複播放 + 付費轉化 | Soundtrack / Anime / Singer-Songwriter |
| **台灣市場動能** | 影視動漫轉化 | Anime / Soundtrack / Mandopop |
| **全球播放基數** | 串流總量 | Pop / Hip Hop / Latin |
| **隨機被動收聽率** | 隨機播放佔比（負向權重） | Trap / Drill / Phonk |

> **關鍵洞察**：Country、Rock、K-Pop、Christian 因 Save Rate 極高 + 黏著度強，位居權重前列；Pop / Hip Hop 雖播放量大，但隨機聽眾多、被動收聽率高，**權重經修正後居中**。

## 💡 三、發行配置建議

### 3.1 Subgenre 自由填寫技巧

| 規則 | 說明 |
|------|------|
| **RouteNote 主+次必填** | 從 39 個合法類別選（如上表） |
| **Subgenre 自由文本** | 可填精確細分風格（如 "Mandopop"、"Phonk"） |
| **Spotify / Apple Music 演算法** | 透過 Subgenre 精準路由到子分類 |

### 3.2 高轉化策略

| 策略 | 推薦場景 |
|------|----------|
| **高 Save Rate 優先** | Country / Rock / K-Pop / Christian（長尾穩定） |
| **亞洲轉媒體** | Anime / Soundtrack / Mandopop（影視流量） |
| **全球破圈** | Latin / Pop / Hip Hop（基數大） |
| **長尾功能性** | Ambient / Lo-fi / Instrumental（Sleep / Focus 場景） |
