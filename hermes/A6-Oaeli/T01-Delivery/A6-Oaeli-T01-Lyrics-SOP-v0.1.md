## 2. 写作工艺约束（继承 Sound DNA + Forbidden + Reference + Visual LOCKED）

### 2.1 Scene 锚定

- 时间：清晨（madrugada / alba 边界 — 城市已醒，she 未醒）
- 空间：Madrid 公寓窗前（女性背影，看城市远景）
- 情绪：intimate solitude（不是 drama，是清晨的微孤独 — 城市噪音白 + 静默自己）
- 韵律：8 音节 / verse（西语 hexasyllabic / octosyllabic 传统）

### 2.2 Forbidden（继承 LOCKED）

- 不写浓 Reggaeton 节拍意象
- 不写 Trap Influence
- 不写 Flamenco / 探戈抒情老套意象
- 不写 Old-school 西语 Vocal Timbre

### 2.3 Reference 风格锚（继承 LOCKED）

- Rigoberta Bandini：现代西语都市抒情代表
- Natalia Lacunza：Bedroom / Indie 西语
- Cariño：现代派 + 实验边界
- Veintiuno：Indie Pop 西班牙代表

### 2.4 Visual 锚（继承 LOCKED）

- 摄影 / Mediterranean 暖色 / 女性背影站立窗前
- tag line 锁定 = "Canta lo que no se dice al despertar"
