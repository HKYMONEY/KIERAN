# Veloria Senda Brand Bible v1.0 (LOCKED 2026-07-14)

> **本文件 = 永久 + 慢演化层（multi-artist-label-playbook 10-correction #3）**。Track 级 SOP 全部写在 Track Playbook，**不入本文件**。
> **角色**：A8 = 拉美 / World "编辑名片 + 文化切片" 长期艺术家（与 A6 Oaeli "Suno 主力走量" 互补，量质分工 LOCKED 7-14）。
> **区分**：A5 Veloria (黄昏 / Ambient Pop) ≠ A8 Veloria Senda (Latin Folk / World)。两个独立 artist / 独立 Spotify Profile / 独立 Brand Bible，**严禁合并**。

---

## §0. Core Positioning（永久）

### 0.1 One-line statement
**"Veloria Senda = 她在高地早市的拂晓里，唱出远方祖母的名字。"**

### 0.2 4 设计原则（永久）
1. **拉丁美洲 = 高地与海岸的呼吸**，不是西班牙本部的都市黄昏
2. **每个 Track = 1 个拉美生活切片**（早市 / 信件 / 山路 / 海岸小镇）
3. **女性嗓音 = 信使**，不抢民俗乐器的叙事权
4. **纪录感视觉优先**：单帧摄影 + 民俗物件 + 高地蓝远景

### 0.3 4 反对项（永久）
1. **反对 Trap / Reggaeton 满拍** → 与 Oaeli 走量的拉丁主盘拉开
2. **反对 EDM beat drop** → World Music 边界
3. **反对多语种混声（es+en+fr）** → 保持 es 单一叙事纯净度
4. **反对品牌合作限定文案入 Track lyric** → Track 必须能独立发行（Lyrics Vault §3.6 LOCKED）

## §2. Sound DNA（慢演化，本节为 v1.0 区间 LOCKED 候选，7-13 区间 LOCKED 红线）

### 2.1 Sound 层 5 项（LOCKED 候选区间）

| # | 字段 | LOCKED 区间 | 锁定理由 |
|---|---|---|---|
| 1 | **BPM** | **90-108 BPM** | Mercado 早市人潮节奏；< Oaeli 80-95 但仍 Latin 系；避 Reggaeton 95-110 模板 |
| 2 | **Key 池** | **Dorian + Mixolydian + Pentatonic** | Latin Folk 异国调性；与 Oaeli 共享 Dorian 根，但不锁单值 |
| 3 | **传统乐器权重** | **25-35%（不点名细分乐器）** | Suno Folk/World "模拟感" 工艺对策（v0.2 §3.0.2 LOCKED）；Prompt 用泛化词 |
| 4 | **人声比** | **60-75% Female + 25-40% Male Chorus** | Latin Folk 男女合唱模式（参考 Lafourcade Casa / Lilián Kvien） |
| 5 | **Suno Vocal 语种** | **es-MX 女主 + es-PE/CL 合唱** | Mercado 在墨西哥-秘鲁-智利均 = Suno 西语 4 大样本池内 |

### 2.2 Forbidden 层 5+1 项（永久）

| 5 | 🚫 多语种混声（es+en+fr）| 保持 es 单一叙事纯净度 |
| 6 | 🚫 品牌合作限定文案入 Track lyric | Track 必须能独立发行（v0.2 §3.6 LOCKED）|

## §3. Lyric DNA（慢演化）

### 3.1 语言锁定

- **Track 1-5**：**es 95% + en 5%**（en 仅用作远方信末签名 / 标题落款）
- **Track 6+**：可逐步引入 **Quechua 词缀 / Nahuatl 词缀**（Andean / Mesoamerican 文化锚），但 es 仍 ≥ 90%

### 3.2 押韵协议（LOCKED）

- **AABB 严押韵**（与 Oaeli 半押韵 LOCKED 区分，Latin Folk 偏传统谣曲韵律）
- 每行 8 音节（octosílabo，参考西班牙黄金世纪谣曲 romancero 传统）
- 段尾词必须回归主韵脚（4 段共用同一 rhyme scheme）

### 3.3 叙事结构（永久）

- **核心叙事锚** = 信（carta）
- **视角** = 第一人称少女写给远方祖母
- **场景** = 拉美山地小镇 Mercado 早市 / Andes 高地 / 海岸小镇 / 长路（camino）
- **情感双轨**：alegría compartida（共同庆祝）+ nostalgia（远方怀念）

### 3.4 主题禁区（永久）

- 🚫 偶像崇拜 / 粉丝迷恋语态
- 🚫 都市酒吧 / 夜店 / clubbing 场景
- 🚫 政治 / 抗议叙事（与"编辑名片"定位冲突，避 Spotify 政治池）
- 🚫 露骨情色 / 性暗示（RouteNote 平台审核红线）

## §12. Lessons Learned

- ❌ **不要把 A5 Veloria 与 A8 Veloria Senda 合并**（路线图 LOCKED 严令，独立 Spotify / RouteNote Profile）
