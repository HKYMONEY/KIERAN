## 模块目录

| # | 模块 | 目录 | 频率 |
|---|---|---|---|
| 01 | Lyrics Lab | `01-lyrics-lab/` | Daily |
| 02 | Mood Library | `02-mood-library/` | Weekly |
| 03 | Trend Radar | `03-trend-radar/` | Daily |
| 04 | Artist DNA Evolution | `04-artist-dna/` | Track 发行后 |
| 05 | Prompt Research Lab | `05-prompt-lab/` | Suno 输出即录入 |
| 06 | Reference Music Library | `06-reference-library/` | Weekly |

| 08 | Audience Memory | `08-audience-memory/` | Track 发行后 |
| 09 | Artist Philosophy | `09-philosophy/` | Quarterly 校准 |
