```bash

  --track-id "<track-name>" --offset D7

# D+30 中期数据

# D+90 长期数据 + DNA 更新
python3 ... --offset D90
```

**Hermes 提醒**: Hermes 在 Track 发行后 D+5 / D+28 / D+88 自动 Telegram 提醒 Kieran "建议手动跑 Performance Review".

### Review 字段 (录入 [[08-audience-memory]])

| 维度 | 字段 |
|---|---|
| Stream | total / unique / completion_rate / skip_rate |
| Save | save_count / save_rate |
| Editorial | playlist_added (哪些 / 何时 / playlist_id) |
| Comments | top 10 共鸣句 + 关键词云 |
| DNA Impact | 是否触发 [[Weini-Brand-Bible-v0.3]] § 1 Slow Evolution 字段更新 |

### DNA 更新规则 (Permanent)

- **D+90 Save rate < 1%**: 该 Track 流派 / 主题 / BPM 标记 "需复盘", 不直接推翻 Bible
- **D+90 Save rate 1-3%**: 维持 Bible 默认值
- **D+90 Save rate 3-5%**: 标记该 Track 流派为 "Optimized" (Bible Slow Evolution 优先级 +1)
- **D+90 Save rate > 5%**: 标记 "Hit", 拆解为什么成功, 反向更新 Reference Library

## 12. Lessons Learned ⭐ NEW v0.3

> **目的**: 每章末尾沉淀经验, Hermes 越来越聪明.

### Music DNA Lessons

- **不要写死单 Track BPM**: 72-78 BPM 是 Brand Establishment 区间, 但 Track 1=74 / Track 2=76 这种具体数字是 Release Planning 范畴, 不进 Bible (Kieran 2026-07-11)
- **不要把 Lyric 168 char 当 SOP**: Suno 当 inspiration 处理, 0% 服从. ≥ 400 char / 8 段触发 100% 服从 (playbook v4.7)
- **不要用 "X minutes length" Style 字段**: Suno 完全忽略. 用 [Outro: long fade out] + Style Influence % 控制时长

### Vocal DNA Lessons

- **不要用描述性词 (intimate)**: Suno 不识别. 用可执行词 (whisper / soft breathy / emotional delivery)

### Visual DNA Lessons

- **不要做"塑料感"民族器乐封面**: Suno 训练集自动加古筝 / 箫污染, Negative Prompt 8 行压制即可, 但封面提示词不要触发

### Distribution Lessons

- **不要默认 Alternative 主曲风**: Spotify Editorial 路由断, 改成 Pop 主 + World 次 (RouteNote 39-list 实证 2026-07-01)

### AI Tool Lessons

- **不要追 Suno "X minutes length"**: 完全无效, 浪费 1 轮抽卡
- **不要假设 5 项中文测试通过**: Kieran 确认 Suno 付费不换, 跳过测试直接进 Track SOP, 但每个 Track 必须 Whisper transcript 验证
