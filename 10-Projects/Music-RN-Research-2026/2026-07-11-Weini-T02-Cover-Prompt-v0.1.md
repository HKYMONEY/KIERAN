# 微霓 Track 02 封面提示词 v0.1 (3 候选)

**主题**: 窗与灯 (T02 LOCKED)
**意象池**: 窗 / 灯 / 雨 / 玻璃 / 影子
**适用工具**: Ideogram 2.0 (免费 50/天, 3000x3000 标准) / Midjourney v6 / Flux

---

## Brand Bible 视觉硬约束 (引用)

### Permanent 必须包含

| 维度 | 锁定值 |
|---|---|
| **核心元素** | 月 / 雾 / 风 / 烟 / 山 / 水 / 灯 / 雁 (8 选 N, T02 选窗/灯/雨/玻璃/影子) |
| **Camera** | 长焦 / 浅景深 / 胶片颗粒 / 雾化光 / 柔焦 / 电影感构图 |
| **色彩** | 月白 / 雾蓝 / 墨灰 / 深青 / 暖金 (点缀) |
| **禁止** | ❌ 高饱和霓虹 / ❌ 卡通萌系 / ❌ K-pop 偶像妆造 / ❌ 二次元角色封面 |

### T02 专项 (基于 Lyrics v0.3)

| 元素 | 母题 | 视觉表达 |
|---|---|---|
| 窗 | 等待 / 凝视 | 公寓窗框 (居中或偏左), 窗台 |
| 灯 | 陪伴 / 孤独 | 桌上一盏暖灯 (金黄色), 玻璃灯罩 |
| 雨 | 时间 / 流过 | 玻璃上的雨滴 / 雨痕 (浅景深虚化) |
| 玻璃 | 距离 / 隔 | 窗玻璃 + 反射的窗内灯 |
| 影子 | 自己 / 留白 | 灯照在桌面的影子 / 窗边剪影 |

## 候选 A: 「窗前静物」(主推)

**风格**: 极简室内静物 + 暖金点缀
**重心**: 桌 + 灯 + 窗 (远处虚化)

### 中文意象 (给 Ideogram)

```
长焦镜头特写一张公寓木桌, 桌上一盏暖金色玻璃灯, 灯罩散发柔光,
桌面上有一杯凉茶, 茶汤微金, 桌面有雨滴溅落的水痕,
背景是一扇窗户, 窗外是模糊的蓝灰色城市夜景, 雨丝斜飘,
浅景深, 胶片颗粒, 35mm 电影感, 月白 + 雾蓝 + 暖金点缀,
极简, 留白, 静谧, no people, no text, no logo,
3000x3000 square composition
```

### English Prompt (Midjourney / Flux)

```
Long-focus 35mm photograph of a quiet apartment wooden desk at night,
a single warm amber glass lamp on the table casting soft light,
a half-cold tea cup with golden liquid, raindrop splatters on wood surface,
background: blurred window with rain streaks and soft bokeh of distant
blue-grey city lights, shallow depth of field, cinematic film grain,
muted palette of moonlit white, fog blue, ink grey with warm amber accent,
minimalist, contemplative, intimate, solitary, contemplative mood,
no people, no text, no logo, no face --ar 1:1 --s 750 --style raw
```

### Negative Prompt

```
high saturation, neon colors, cartoon, anime, k-pop, idol,
face, people, text, logo, watermark, busy composition,
bright daylight, warm sunshine, summer vibe, tropical
```

## 候选 B: 「雨中窗」(窗框为视觉中心)

**风格**: 窗外视角 + 雨滴 + 暖光反射
**重心**: 玻璃窗 + 雨滴 + 室内暖光

### 中文意象

```
从窗外视角看公寓窗玻璃, 玻璃上有 100+ 雨滴, 雨滴被室内暖金色灯光
照亮成琥珀色, 雨痕缓慢流下, 玻璃窗后虚化的室内有一盏亮着的灯,
窗外是深青色夜雨, 浅景深聚焦在玻璃中段雨滴, 胶片颗粒,
电影感构图, 月白 + 雾蓝 + 深青 + 暖金点缀,
静谧, 等待, no people, no text, no logo, 3000x3000
```

### English Prompt

```
View from outside a window at night during rain,
the glass covered with 100+ raindrops, each raindrop illuminated
from inside by warm amber lamp light turning into amber pearls,
slow rain streaks flowing down, soft blurred interior behind
the glass with single warm lamp visible through fogged glass,
deep teal night outside, shallow depth of field focused on mid-glass,
cinematic film grain, muted palette of moonlit white, fog blue,
deep teal with warm amber accent, contemplative, intimate,
no people, no text, no logo --ar 1:1 --s 750 --style raw
```

### Negative Prompt

## 候选 C: 「灯下剪影」(深夜意境, 最契合 A1 听感)

**风格**: 灯下剪影 + 窗框 + 雨
**重心**: 人物剪影 (背影/侧影) + 灯 + 窗
⚠️ 注意: Brand Bible § 2.4 禁止 "高饱和 / 偶像妆造", 但**剪影不等于人物特写**, 模糊剪影可接受

### 中文意象

```
深夜公寓窗前, 一个模糊的背影剪影坐在椅子上, 面对窗户,
窗外是雨夜的城市 (虚化蓝灰), 旁边桌上一盏暖金色台灯亮着,
灯光照出剪影边缘金色轮廓, 桌面有茶的蒸汽,
胶片颗粒, 长焦, 浅景深, 电影感构图, 月白 + 雾蓝 + 暖金,
极简, 留白, 静谧, 等待, no face, no text, no logo,
3000x3000
```

### English Prompt

```
Long-focus cinematic photograph at night, a blurred silhouette
of a person sitting on a chair facing a rain-streaked window,
back view, no face visible, beside a warm amber table lamp
casting golden rim light on the silhouette edge,
faint tea steam rising from cup on the table,
rainy cityscape blurred through window, fog blue and ink grey
background with warm amber accent, 35mm film grain,
contemplative, solitary, intimate, minimalist,
no face detail, no text, no logo --ar 1:1 --s 750 --style raw
```

### Negative Prompt

```
high saturation, neon, cartoon, anime, k-pop, idol makeup,
face close-up, portrait, frontal face, bright daylight,
sunshine, tropical, busy composition, multiple people
```

## 3 候选对比

| 维度 | A 窗前静物 | B 雨中窗 | C 灯下剪影 |
|---|---|---|---|
| **元素覆盖** | 窗/灯/雨/玻璃/影子 (5/5) | 窗/灯/雨/玻璃 (4/5) | 窗/灯/雨/影子 (4/5) |
| **极简度** | 高 | 中 | 中 |
| **剪影 / 人物** | 无 (纯静物) | 无 (无人物) | 模糊剪影 |
| **Brand 调性契合** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐ |
| **中文 Ideogram 命中** | 高 | 高 | 中 |
| **暖金点缀强度** | 中 | 高 | 中 |
| **3000x3000 标准** | ✓ | ✓ | ✓ |

**推荐**: **候选 A 窗前静物** (主推)
- **纯静物, 无人**, 完全规避 Brand Bible 风险
- 5/5 元素全覆盖
- 中文 Ideogram 命中率高

**备选**: **候选 B 雨中窗** (如果想要更视觉冲击, 雨滴 amber 琥珀感强)

**候选 C 灯下剪影** — 我**不推荐**, 原因是剪影虽然不直接违反 Brand Bible 永久禁止, 但
- Ideogram / Midjourney 经常把人脸/身体画得很清楚, 模糊剪影难控制
- 风险: 触发 "高饱和偶像妆造" 二次元封面 等禁止元素
- 如果你想要 "有人物" 的意境, 建议用候选 B 加人工合成, 不要纯 AI 生成
